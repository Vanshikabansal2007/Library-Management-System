# 📚 Library Management System

A simple Library Management System developed using Python and Object-Oriented Programming (OOP).

## Features

- Add Books
- Add Members
- Display Books
- Display Members
- Search Book
- Issue Book
- Return Book

## OOP Concepts Used

- Classes & Objects
- Encapsulation
- Composition

## Project Structure

- main.py
- book.py
- member.py
- library.py

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- VS Code

## How to Run

```bash
python main.py
```
